## 📁 Project Structure

```
IRP Chatbot/
├── package.json                 # Root package.json for concurrent dev setup
├── README.md                    # This file
│
├── backend/                     # Node.js/Express API server
│   ├── package.json            # Backend dependencies
│   ├── server.js               # Main server entry point
│   ├── env.example             # Environment variables template
│   ├── config/
│   │   └── db.js              # Database connection configuration
│   └── routes/
│       └── api.js             # API route handlers
│
└── frontend/                   # React application
    ├── package.json           # Frontend dependencies
    ├── tailwind.config.js     # TailwindCSS configuration
    ├── postcss.config.js      # PostCSS configuration
    ├── public/
    │   ├── index.html         # HTML template
    │   └── manifest.json      # PWA manifest
    └── src/
        ├── index.js           # React entry point
        ├── index.css          # Global styles and Tailwind imports
        ├── App.js             # Main application component
        ├── services/
        │   └── api.js         # API service layer
        └── components/
            ├── SchemaPanel.js      # Database schema explorer
            ├── QueryInput.js       # SQL query input component
            ├── ResultsTable.js     # Query results display
            └── SuggestedQueries.js # Example queries panel
```
